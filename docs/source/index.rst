Welcome to eml-pytorch's documentation!
=======================================

**eml-pytorch** is a hardware-efficient Exp-Minus-Log (EML) operator for PyTorch,
providing a composable primitive ``eml(x, y) = exp(x) - log(y)`` with full autograd support
and optional Triton GPU acceleration.
